### Mblog (mtons blog)开源免费的Java多人博客系统

### 技术选型：

* JDK8
* 数据库MySQL
* 主框架 (Spring、SpringMVC、Spring-data-jpa）
* 安全权限 Shiro
* 搜索工具 Lucene
* 缓存 Ehcache
* 视图模板 Velocity
* 其它 Jsoup、fastjson
* jQuery、Seajs
* Bootstrap 前端框架
* UEditor/Markdown编辑器
* font-wesome 字体/图标
